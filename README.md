# [CREMA] Infrastructure

이 문서는 [CREMA]의 인프라 구성과 운영 방법을 안내합니다.

## 📜 개요 (Overview)

본 인프라는 [CREMA]을 [GCP] 환경에 배포하고 안정적으로 운영하는 것을 목표로 합니다.

- **Cloud Provider**: `GCP`
- **Orchestration**: `Kubernetes`
- **CI/CD**: `Jenkins` / `ArgoCD`
